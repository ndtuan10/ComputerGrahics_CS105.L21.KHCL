Hướng dẫn cài đặt và sử dụng đồ án
1. Cài đặt
	1/ Python (version 3.x). https://www.python.org/downloads/
	2/ IDE ví dụ Visual Studio Code. https://code.visualstudio.com/download
2. Hướng dẫn sử dụng
	Sau khi unzip file “DoAn_18520186-18521027-18520922”, tại đây có hai folder là “Sources” và “Docs”. Ta vào folder “Sources” chứa toàn bộ source code của đồ án, chọn và click vào file index.html để hiển thị ra giao diện mô phỏng hình học 3D trên web, với title là “Demo Computer Graphics”.
	Tại góc phải màn hình, ta sẽ thấy lần lượt có 5 chức năng chính để thao tác và tùy chỉnh. Ví dụ để chọn objects và material tương ứng, ta sẽ sử dụng tính năng “Geometries”. Tương tự ta có thể chọn và thay đổi các tính năng khác của object, camera và light bằng giao diện dat.GUI.
	Để thuận tiện trong việc theo dõi cách sử dụng đồ án, ta có thể theo dõi video demo mà chúng tôi đã quay sẵn.
